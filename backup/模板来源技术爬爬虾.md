🚀 Gmeek 是什么？
Gmeek 是一个由国内开发者（GitHub: Meekdai）开源的静态博客生成器，专为追求极简、轻量、易用的用户设计。
它类似 Hexo、Hugo，但更注重简洁配置和快速部署，适合不想折腾复杂功能、只想专注写作的用户。

✨ 主要特点
极简设计

默认主题干净、响应式，适合文字为主的博客。

无冗余功能，专注文章展示。

轻量快速

依赖少，生成静态页面速度快。

页面资源小，访问体验流畅。

易于上手

配置简单，只需修改少量设置（如 config.yml）。

支持 Markdown 写作，自动生成文章列表、标签页等。

部署方便

可一键部署到 GitHub Pages、Vercel、Netlify 等平台。

📦 技术栈
基于 Node.js（或类似工具链，具体看版本）。

使用模板引擎生成静态 HTML。

支持 Sass/CSS 预处理。
